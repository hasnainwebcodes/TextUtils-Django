# TextUtils
This is a simple web that manipulates text and gives desired output text. This site is made with Django to understand and leverage Django core concepts and make a useable and beneficial product. It has no Database Integration only core Django and pure python with logic. In order to understand the compatibility of the Django with real life problems faced by the software industry, this project helps to understand the solution of those problems in limited sense.
